# Ejecutar en desarrollo

1. clonar el repositorio
2. Ejecutar

```
$ npm install

$ npm run start:dev
```

3. Tener Nest CLI instalado

```
$ npm i -g @nestjs/cli
```

4. Levantar la base de datos con docker

```bash
$ docker-compose up
```

5. Recontruir la base de datos

```bash
localhost:3000/api/v2/seed
```

## Starck usado

- NestJS
- MongoDB
- Dockers
- Mongoose
